## github에 코딩파일 수정 후 업로드
1. 업로드할 파일만 + 눌러서 staged Changes로 이동
2. Message 입력칸에 어떤 것이 변경되었는지 입력하고 commit 버튼 클릭
3. Sync Changes 버튼 클릭하면 내 컴퓨터에서 github로 수정된 코드 전송
4. 저장저장
